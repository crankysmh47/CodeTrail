# Changelog

## 0.1.0 - 2026-07-15

- Added a local VS Code question-to-trail workflow for GNU C workspaces.
- Added Tree-sitter C parsing through packaged WebAssembly assets.
- Added Linux scheduler enrichment for designated initializers, scheduler-class registration, function-pointer dispatch, and configuration guards.
- Added deterministic, relationship-aware search with unique results, visible match reasons, and bounded identifier typo recovery.
- Added bounded graph traversal with a cross-file route followed by within-file symbol paths.
- Added function CodeLens, an editor context action, and `Alt+Shift+T` for exact symbol discovery.
- Added a compact VS Code-native panel with persistent search and explicit empty-state recovery.
- Added worker-thread analysis, validated gzip snapshots, source navigation, and optional Clang capability reporting.
- Added the pinned Linux scheduler demo, gold tests, performance budgets, and an installable VSIX build.

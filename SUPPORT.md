# CodeTrail support

Use [GitHub Issues](https://github.com/crankysmh47/CodeTrail/issues) for reproducible bugs, installation problems, and focused feature requests.

Include the CodeTrail version, VS Code version, operating system, indexed file count, the command or search that failed, and the exact error message. A small public C fixture is helpful when you can share one. Do not attach private source, credentials, full workspace snapshots, or proprietary logs.

CodeTrail has no hosted account or paid support channel. Security-sensitive reports should follow [SECURITY.md](SECURITY.md), not the public issue tracker.
